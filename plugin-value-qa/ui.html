값 검수 · 디자인 ↔ 개발

피그마 정답값을 라이브로 읽어 개발·퍼블리싱 화면과 대조. 아래 단계를 순서대로.
0 측정 도구 준비 — 처음 한 번만▾
검사할 화면(퍼블리싱·개발 둘 다 됨)에서 클릭 한 번으로 값을 뽑는 북마크를 만들어요.
복사한 뒤: 북마크바 우클릭 → 페이지 추가 → 이름 📐 화면 측정, 주소에 붙여넣기.
또는 같은 폴더의 setup.html을 크롬으로 열어 버튼을 북마크바로 드래그.
1 정답 읽기 (피그마) ▾
피그마에서 정답이 될 화면 프레임을 선택하세요.
2 개발·퍼블리싱 측정 넣기 ▾
← 먼저 1단계(정답 읽기)를 끝내세요.
여기에 measure-*.json을 끌어다 놓거나 클릭
북마크 '📐 화면 측정' 클릭하면 나오는 파일 (퍼블리싱·개발 어느 쪽이든)
3 짝 확인 — 자동으로 맞췄어요. 틀린 것만 손보세요▾
← 1·2단계를 끝내면 자동으로 짝을 맞춰 보여줘요.
4 결과 리포트▾
← 3단계에서 '값 대조하기'를 누르면 결과가 나와요.
🖥 개발화면 이미지 — 선택 · 정답 오른쪽에 나란히▾
← 1단계(정답 읽기) 후 사용할 수 있어요.
개발화면 캡처(PNG) 끌어다 놓기 / 클릭
화면에서 ⌘⇧P → 'Capture full size screenshot'
