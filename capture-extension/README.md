# 화면 측정 + 픽셀 캡처 확장앱 (개발화면 검수용)

**클릭 한 번**에 화면의 **값(JSON)** + **픽셀-정확 전체화면(PNG)**을 받는 크롬 확장앱.
개발화면 검수의 "값 대조 + 나란히 눈검수"에 필요한 두 입력을 한 번에 만들어요. 픽셀 100% 정확(브라우저 실제 렌더 캡처).

## 왜 확장앱인가
브라우저는 보안상 **일반 스크립트(콘솔/북마크)가 실제 화면 픽셀을 못 잡게** 막아요. 픽셀-정확 캡처는
확장앱(또는 DevTools)에서만 됩니다. 이 확장앱은 `chrome.debugger`로 **전체페이지 실픽셀**을 잡아요.

## 설치 (딱 한 번)
1. 크롬 주소창에 **`chrome://extensions`** 입력 → Enter
2. 오른쪽 위 **개발자 모드(Developer mode)** 켜기
3. **압축해제된 확장 프로그램을 로드(Load unpacked)** 클릭
4. 이 폴더(**`capture-extension`**)를 선택
5. 툴바에 퍼즐 아이콘 → **화면 측정 + 캡처** 확장을 고정(📌)해두면 편해요

## 사용
1. 검사할 **개발화면**을 크롬으로 엽니다 (검수할 폭 = 창 폭. 정답과 같은 폭으로).
2. 툴바의 **📐 화면 측정 + 캡처** 아이콘을 **클릭**.
3. 잠깐 "이 브라우저를 디버깅 중" 배너가 떴다 사라지고(무해), 다운로드 폴더에:
   - `measure-<화면>.json` — 값
   - `screen-<화면>.png` — 픽셀-정확 전체화면
4. 두 파일을 검수에 사용: JSON은 플러그인 ②번에, PNG는 정답 옆 나란히 배치에.

## 안내
- **폭 맞추기**: 창 폭이 곧 측정·캡처 폭이에요. 정답(피그마) 폭과 같게 하세요.
- **로컬 dev(127.0.0.1)** 등 대부분에서 동작. 일부 특수 페이지(chrome:// 등)는 캡처 불가.
- 값 측정 로직은 `plugin-value-qa/collect-core.js` **단일 원본**에서 옵니다(`build-tools.js`가 이 폴더로 동기화 복사). 직접 `collect-core.js`를 고치지 마세요.
- 캡처 없이 **값만** 필요하면 북마크(📐 화면 측정)로도 돼요(설치 불필요).
