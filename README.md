# 개발화면 검수 (Figma 플러그인)

피그마 최종 디자인(정답)과 개발화면 캡처(실제)를 나란히 두고,
다른 부분을 기록·이력관리하는 도구입니다. **외부 통신·비용 없음.**

## 피그마에 올리는 법 (한 번만)
1. Figma **데스크톱 앱**을 엽니다.
2. 아무 파일이나 연 뒤 메뉴 → **Plugins → Development → Import plugin from manifest…**
3. 이 폴더의 **manifest.json** 을 선택합니다.
4. 이제 **Plugins → Development → 디자인 QA 대조** 로 언제든 실행됩니다.

## 지금 되는 것 (1단계 · 관리 뼈대)
- 화면 목록 대시보드 + 상태(미개발/검수중/완료/신규/보류) + 진행도
- 피그마 프레임 연결 → 디자인 자동 추출(PNG)
- 개발화면 캡처 **파일 선택 / 드래그 / 붙여넣기(Ctrl+V)**
- 버전 관리: 재캡처 시 자동 버전 + '새 버전 저장' 수동
- 이슈 이력: 열림→수정요청→개발수정됨→검증완료 (완료도 삭제 않고 서랍 보관)
- 모든 내용은 이 피그마 파일 안에 저장
